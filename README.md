# ChequeMate – Personal Cheque Ledger PWA

A Progressive Web App that helps a user track, manage, and monitor issued cheques offline using local device storage.

## Features

- Installable as a mobile app
- Works fully offline
- Stores data in device (IndexedDB)
- No cloud / no database required
- Fast and lightweight

## Tech Stack

- React
- PWA (Service Worker + Manifest)
- IndexedDB
- Hosted on Vercel

## How to Run

npm install
npm start

## How to Build PWA

npm run build
